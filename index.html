Passo 1

Modelagem de Banco de Dados

Foca no Indicador 1
Ao concluir estas tarefas e anexar o print, você estará concluindo o Indicador 1!

Antes de ir para o computador, precisamos desenhar como as informações se organizam. Para criar esse diagrama conceitual e lógico, você utilizará a ferramenta gratuita draw.io (disponível online em app.diagrams.net).

Faremos um banco de dados para gerenciar uma Escola. Desenhe no draw.io 2 tabelas relacionais para o nosso mini-projeto:

  • Turma: Representa a entidade Turma (ex: "Desenvolvimento Web"). Uma turma pode ter vários alunos.
  • Aluno: Representa a entidade Aluno (com nome e e-mail). Cada aluno pertence a uma turma específica.

Como fazer no Draw.io:

  1. Acesse app.diagrams.net e crie um novo diagrama em branco.
  2. Na barra lateral esquerda, clique em + Mais Formas, marque a opção Relação de Entidades (Entity Relation) e clique em Aplicar.
  3. Arraste duas caixas de tabela (Entidades) para a tela e dê os nomes de turma e aluno.
  4. Adicione os atributos (campos), marcando as Chaves Primárias (PK) e a Chave Estrangeira (FK).
  5. Conecte as duas tabelas com uma linha de relacionamento de 1 para N (1:N) (onde 1 turma possui N alunos).

Visualização do Diagrama de Referência

O seu diagrama no draw.io deve ficar parecido com o modelo lógico a seguir:

turma
PK id_turma INT (PK)
nome_turma VARCHAR(50)
1 : N
aluno
PK id_aluno INT (PK)
nome VARCHAR(100)
email VARCHAR(100)
FK id_turma INT (FK)
O que são PK e FK? - Primary Key (PK): O código identificador único de cada registro (não se repete). - Foreign Key (FK): A ligação que diz a qual turma aquele aluno pertence. Garante a integridade referencial dos dados.

Passo 2

Preparação do Servidor (Uniform Server)

Foca no Indicador 2
Ao concluir estas tarefas e anexar o print, você estará concluindo o Indicador 2!

Para rodar nossa aplicação PHP e o banco de dados MySQL localmente de forma estável, utilizaremos o Uniform Server (UniServer). Ele é muito mais leve e portátil que o XAMPP.

Passo a Passo de Configuração:

  1. Baixe o Uniform Server Zero (versão portátil) no SourceForge: Download UniServer.
  2. Ele virá em um arquivo compactado (ou auto-extraível). Extraia o conteúdo diretamente no seu disco principal, por exemplo em: C:\UniServerZ.
  3. Abra a pasta C:\UniServerZ e dê dois cliques no arquivo executável Start.exe.
  4. No painel gráfico que abrir, clique nos botões Start Apache e Start MySQL para iniciar os servidores locais.
  5. (Opcional) O UniServer solicitará que você defina uma nova senha para o MySQL no primeiro acesso. Para facilitar nosso projeto local, você pode definir como vazia (apenas dê enter/confirme) ou usar a senha padrão.
Onde colocar os arquivos PHP? No Uniform Server, a pasta padrão do servidor de arquivos web é a www (e não htdocs como no XAMPP). Logo, seus arquivos devem ficar dentro de C:\UniServerZ\www\.

Criação do Banco de Dados Físico (SQL)

Foca no Indicador 3
Ao concluir estas tarefas e anexar o print, você estará concluindo o Indicador 3!

Com o MySQL rodando no UniServer, vamos criar a estrutura do banco físico usando a aba SQL do phpMyAdmin.

Como fazer no phpMyAdmin do UniServer:

  1. No menu do UniServer (ou digitando no navegador http://localhost/phpmyadmin/), abra a interface do phpMyAdmin.
  2. Clique na aba superior chamada SQL.
  3. Copie o código SQL fornecido abaixo, cole no editor do phpMyAdmin e clique em Executar.
SQL Script 
-- 1. Criar o banco de dados
CREATE DATABASE IF NOT EXISTS db_escola 
CHARACTER SET utf8mb4 
COLLATE utf8mb4_unicode_ci;

USE db_escola;

-- 2. Criar tabela de Turmas (DDL)
CREATE TABLE IF NOT EXISTS turma (
    id_turma INT AUTO_INCREMENT PRIMARY KEY,
    nome_turma VARCHAR(50) NOT NULL
);

-- 3. Criar tabela de Alunos (DDL com Chave Estrangeira)
CREATE TABLE IF NOT EXISTS aluno (
    id_aluno INT AUTO_INCREMENT PRIMARY KEY,
    nome VARCHAR(100) NOT NULL,
    email VARCHAR(100) NOT NULL,
    id_turma INT NOT NULL,
    FOREIGN KEY (id_turma) REFERENCES turma(id_turma) 
        ON DELETE RESTRICT
);

-- 4. Inserir dados de teste (DML)
INSERT INTO turma (nome_turma) VALUES 
('Informática para Internet'),
('Desenvolvimento de Sistemas');

INSERT INTO aluno (nome, email, id_turma) VALUES
('Ana Silva', 'ana@email.com', 1),
('Bruno Souza', 'bruno@email.com', 2),
('Raquele Lima', 'raquele@email.com', 1);

Passo 4

Integração do PHP com Banco de Dados

Foca no Indicador 4
Ao concluir estas tarefas e anexar o print, você estará concluindo o Indicador 4!

Com o banco físico criado, faremos a conexão do código da nossa aplicação PHP usando a classe PDO.

Criando a pasta do Projeto:

Navegue até a pasta de arquivos do seu UniServer (C:\UniServerZ\www). Crie uma pasta chamada recuperacao_escola dentro dela. Coloque o código de conexão abaixo em um arquivo chamado conexao.php nessa pasta:

conexao.php 
<?php
$host = 'localhost';
$dbname = 'db_escola';
$username = 'root';
$password = ''; 

try {
    $pdo = new PDO("mysql:host=$host;dbname=$dbname;charset=utf8", $username, $password);
    $pdo->setAttribute(PDO::ATTR_ERRMODE, PDO::ERRMODE_EXCEPTION);
} catch (PDOException $e) {
    die("Erro ao conectar com o banco de dados: " . $e->getMessage());
}
?>

Passo 5

Manipulação de Dados a partir da Aplicação (CRUD)

Foca no Indicador 5
Ao concluir estas tarefas e anexar o print, você estará concluindo o Indicador 5!

Nesta última etapa do projeto, criaremos as duas páginas principais da aplicação para inserir e ler dados do banco:

  • Listagem (index.php): Fará uma consulta SQL com a cláusula INNER JOIN para trazer os dados do aluno junto com o nome correspondente da sua turma.
  • Cadastro (cadastrar.php): Contém um formulário para inserir alunos de forma segura via POST utilizando Prepared Statements.

1. Arquivo de Listagem: index.php

index.php 
<?php
require 'conexao.php';

$sql = "SELECT aluno.id_aluno, aluno.nome, aluno.email, turma.nome_turma 
        FROM aluno 
        INNER JOIN turma ON aluno.id_turma = turma.id_turma";

$stmt = $pdo->query($sql);
$alunos = $stmt->fetchAll(PDO::FETCH_ASSOC);
?>
<!DOCTYPE html>
<html lang="pt-BR">
<head>
    <meta charset="UTF-8">
    <title>Lista de Alunos</title>
    <link rel="stylesheet" href="estilo.css">
</head>
<body>
    <div class="container">
        <h2>Alunos Cadastrados</h2>
        <a href="cadastrar.php" class="btn">Cadastrar Novo Aluno</a>
        
        <table border="1" style="width:100%; border-collapse: collapse; margin-top:20px;">
            <tr>
                <th>ID</th>
                <th>Nome</th>
                <th>E-mail</th>
                <th>Turma</th>
            </tr>
            <?php foreach ($alunos as $aluno): ?>
            <tr>
                <td><?= $aluno['id_aluno'] ?></td>
                <td><?= htmlspecialchars($aluno['nome']) ?></td>
                <td><?= htmlspecialchars($aluno['email']) ?></td>
                <td><strong><?= htmlspecialchars($aluno['nome_turma']) ?></strong></td>
            </tr>
            <?php endforeach; ?>
        </table>
    </div>
</body>
</html>

2. Arquivo de Cadastro: cadastrar.php

cadastrar.php 
<?php
require 'conexao.php';

$turmas = $pdo->query("SELECT * FROM turma")->fetchAll(PDO::FETCH_ASSOC);

// Inserção de dados de forma segura
if ($_SERVER['REQUEST_METHOD'] === 'POST') {
    $nome = $_POST['nome'] ?? '';
    $email = $_POST['email'] ?? '';
    $id_turma = $_POST['id_turma'] ?? '';

    if (!empty($nome) && !empty($email) && !empty($id_turma)) {
        $stmt = $pdo->prepare("INSERT INTO aluno (nome, email, id_turma) VALUES (:nome, :email, :id_turma)");
        $stmt->execute([
            ':nome' => $nome,
            ':email' => $email,
            ':id_turma' => $id_turma
        ]);
        header("Location: index.php");
        exit;
    }
}
?>
<!DOCTYPE html>
<html lang="pt-BR">
<head>
    <meta charset="UTF-8">
    <title>Cadastrar Aluno</title>
    <link rel="stylesheet" href="estilo.css">
</head>
<body>
    <div class="container">
        <h2>Cadastrar Novo Aluno</h2>
        <form action="cadastrar.php" method="POST">
            <p>Nome: <input type="text" name="nome" required></p>
            <p>E-mail: <input type="email" name="email" required></p>
            <p>Turma: 
                <select name="id_turma" required>
                    <option value="">Selecione</option>
                    <?php foreach ($turmas as $turma): ?>
                        <option value="<?= $turma['id_turma'] ?>"><?= htmlspecialchars($turma['nome_turma']) ?></option>
                    <?php endforeach; ?>
                </select>
            </p>
            <button type="submit">Salvar Aluno</button>
            <a href="index.php">Voltar</a>
        </form>
    </div>
</body>
</html>
